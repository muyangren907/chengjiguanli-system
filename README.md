统计学生成绩是我们日常工作中不可缺少的一部分，它为教学与学习提供参考，它的工作量又占用老师们的大量时间与精力。网上统计成绩的软件很多，与教师工作习惯相同、又能提供教师需要的数据的软件却少之又少，为让了统计成绩变得更优雅，才有了这个小学生成绩统计系统。前端采用H-ui框架，后端采用ThinkPHP框架开发。
